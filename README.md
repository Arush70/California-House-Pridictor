# ECMM422J Coursework 2 — California Housing Price Prediction

Predicting median house value for California districts using clustering
(K-Means + KNN) and four regressors (LR, RFR, SVR, MLP).

## Structure
- `notebooks/Training_Models.ipynb` — full pipeline
- `models/` — saved trained models
- `figures/` — plots used in the report
- `report/` — LaTeX report source

## Data
California Housing dataset (1990 Census).
Source: https://github.com/ageron/data/raw/main/housing.tgz

## How to run
Open the notebook in Colab and run top to bottom.
